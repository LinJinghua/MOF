1.RMSD 
MSD (Mean Square Displacement) ，均方位移，其定义式如下：  
$$MSD=<|r(t) - r(0)|^2>=1/N \sum_1^N |(r(t) - r(0))|_^2$$
其中 <> 是对组内的所有原子进行平均。
